## Installation

- ### [BetterDiscord](https://github.com/BetterDiscord/BetterDiscord)

  - download the theme file.
  - place it in the BetterDiscord themes folder || folder directory : %appdata%/betterdiscord/themes.
  - enable **soon** in the themes tab.
